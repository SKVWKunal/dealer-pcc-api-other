# Website Background & Context

## Skoda | Volkswagen - Dealer Portal System

### Overview
This is an internal corporate web platform designed for Skoda and Volkswagen dealerships and service centers. The website provides a centralized hub for dealer operations, technical training registration, survey management, and API integration services. The interface maintains a corporate-clean aesthetic with emphasis on mobility, safety, and digital integration.

---

## Visual Identity

**Logo:** Skoda | Volkswagen  
**Brand Position:** Joint branding representing both automotive manufacturers  
**Design Language:** Corporate-clean, professional, mobility-focused

---

## Navigation Structure

The top navigation bar includes the following sections:

### Primary Navigation Items

1. **API Registration**
   - Purpose: Register and manage API access for dealer systems
   - Target: Technical administrators and IT personnel
   - Key function: Enable integration between dealer systems and manufacturer platforms

2. **MT Meet Registration (Master Technician)**
   - Purpose: Registration portal for Master Technician meetings and training sessions
   - Target: Senior technicians and workshop managers
   - Key function: Schedule and track technical training participation

3. **Workshop System Survey**
   - Purpose: Collect feedback on workshop management systems and tools
   - Target: Workshop managers and service advisors
   - Key function: Gather operational insights for system improvements

4. **Warranty Survey**
   - Purpose: Survey platform for warranty claim processes and experiences
   - Target: Service personnel handling warranty claims
   - Key function: Monitor and improve warranty handling efficiency

5. **Technical Awareness Session Survey**
   - Purpose: Feedback collection for technical training sessions
   - Target: Technicians and service staff attending technical programs
   - Key function: Assess training effectiveness and identify improvement areas

6. **Database**
   - Purpose: Centralized access to technical documentation and dealer resources
   - Target: All authorized dealer personnel
   - Key function: Repository for manuals, bulletins, and reference materials

7. **Gallery**
   - Purpose: Event photography and visual documentation
   - Target: All stakeholders
   - Key function: Showcase training events, dealer meetings, and corporate activities

8. **About Us**
   - Purpose: Information about the dealer support program and organizational structure
   - Target: New users and general visitors
   - Key function: Provide context about the platform and its objectives

9. **FAQ**
   - Purpose: Frequently asked questions about the website and its services
   - Target: All users
   - Key function: Self-service support for common queries

10. **Contact**
    - Purpose: Communication channel for support and inquiries
    - Content includes:
      - Helpline number for immediate assistance
      - Email address for general inquiries and support requests
    - Target: Users requiring direct assistance
